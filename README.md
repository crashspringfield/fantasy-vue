# fantasy-vue
A wrapper for integrating [fantasy-land](https://github.com/fantasyland/fantasy-land) to [Vuejs](http://vuejs.org)

## Installation

```
npm i fantasy-vue
```

## Usage

`main.js`:

    import Vue from 'vue'
    import FantasyVue from 'fantasy-vue'

    Vue.use(FantasyVue)

in your project:

    this.$fantasy.map
